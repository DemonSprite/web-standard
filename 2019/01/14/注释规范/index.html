注释规范

该规范目前主要用于vue项目，也可以部分适用普通前端项目
所有单行注释 的双斜杠 后面要加两个空格
所有多行注释 /* ( 至少两个星号)开头/

vue 业务组件注释

vue 公用组件注释

方法，类或者函数的注释

（1）如果即没有入参也没有返回值，用单行注释方法说明即可

（2）如果有入参或者返回值，用多行注释

4.在函数或者方法内部对复杂逻辑块用单行注释说明
5.变量用单行注释进行说明，写在变量的右面
